# Variables CSS
--bg-main:      Color de fondo principal de la página
--bg-surface:   Color de fondo para cards y superficies
--bg-glass:     Fondo semi-transparente para efectos glass-morphism
--text-main:    Color de texto principal
--text-muted:   Color de texto secundario/desvanecido
--primary:      Color primario (azul/cian) - botones, enlaces, acentos
--primary-glow: Versión con transparencia del color primario
--border:       Color de bordes
--shadow:       Color de sombras
--nav-h:        Altura del header (68px)
--radius:       Radio de bordes (3px)
--container:    Ancho máximo del contenedor (1200px)
--transition:   Duración y tipo de animación estándar
--font-tech:    Fuente tecnológica (Orbitron)
--font-body:    Fuente de cuerpo (Exo 2)
# Funcionalidad JS
### Tema
se permite el cambio de tema claro a oscuro y viceversa haciendo click en alguno de los botones 🌙 / ☀️
### Filtro de categorias 
Se muestran por categoría (Todos, Tecnología, Deporte, Energón)
# Instrucciones de Uso
El sitio está dividido en cuatro secciones principales: la página de inicio presenta un resumen con las noticias destacadas, la sección de artículos te permite explorar todas las noticias disponibles filtradas por categorías (Tecnología, Deporte y Energón) o buscar temas específicos usando el buscador, cada artículo individual te muestra el contenido completo con información adicional en la barra lateral, y la página "Sobre Nosotros" te cuenta la historia, misión y valores del medio. En cualquier momento puedes cambiar entre tema claro y oscuro haciendo clic en el icono de la luna o sol en la esquina superior derecha del header.
